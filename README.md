# 🛒 Olist E-Commerce Sales Analysis Dashboard

A complete Business Intelligence project built using **Power BI** to analyze sales performance, product performance, and seller insights using the Brazilian Olist E-Commerce dataset.

---

# 📌 Project Overview

This project transforms raw e-commerce data into an interactive Power BI dashboard that helps business stakeholders monitor performance and make data-driven decisions.

The project focuses on:

- Executive sales overview
- Product performance analysis
- Seller performance analysis
- Revenue trends
- Freight cost analysis

---

# 🎯 Business Objectives

This dashboard answers important business questions such as:

- How much revenue has the business generated?
- Which product categories generate the highest revenue?
- Who are the top-performing sellers?
- How much does shipping cost impact the business?
- What is the average product price?
- How have sales changed over time?

# 🧹 Data Preparation

The dataset was cleaned and transformed using **Power Query**.

Cleaning process included:

- Data type validation
- Missing value inspection
- Duplicate checking
- Relationship validation
- Column renaming
- Category translation (Portuguese → English)

---

# 🗂 Data Model

A relational star-like model was created using multiple tables:

- Orders
- Order Items
- Customers
- Products
- Sellers
- Payments
- Reviews
- Product Category Translation

Relationships between all tables were manually reviewed and validated.
